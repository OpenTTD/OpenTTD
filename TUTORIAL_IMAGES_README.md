# Tutorial Images Guide

## 图片放置位置

将 18 张教程图片放在以下任一位置（按优先级）：

1. **用户目录** (推荐): `~/.openttd/tutorial_images/` 
   - Windows: `C:\Users\<你的用户名>\Documents\OpenTTD\tutorial_images\`
   - Linux: `~/.openttd/tutorial_images/`
   - macOS: `~/Documents/OpenTTD/tutorial_images/`

2. **游戏安装目录**: `<OpenTTD安装路径>/tutorial_images/`

## 图片命名规则

```
tutorial_1.png  - 基本操作界面
tutorial_2.png  - 道路建设（或组合图）
tutorial_3.png  - 道路建设步骤2（如果分开）
...
tutorial_8.png  - 道路建设最后步骤
tutorial_9.png  - 铁路建设
tutorial_10.png - 铁路建设步骤2
tutorial_11.png - 铁路建设步骤3
tutorial_12.png - 桥梁示例
tutorial_13.png - 隧道示例
tutorial_14.png - 机场/飞机
tutorial_15.png - 码头/船只
tutorial_16.png - 综合示例1
tutorial_17.png - 综合示例2
tutorial_18.png - 综合示例3
```

## 图片要求

- **格式**: PNG（支持 8位/24位/32位色深）
- **尺寸**: 建议宽度 300-600px，高度根据内容自适应
- **文件大小**: 单张建议不超过 500KB
- **内容**: 清晰的游戏截图，裁剪掉无关内容

## 如何截图

1. 启动 OpenTTD
2. 按 `Ctrl+S` 截图（保存在 `screenshot/` 目录）
3. 使用图片编辑软件裁剪到合适尺寸
4. 重命名为 `tutorial_1.png` 等并放到 `tutorial_images/` 目录

## 临时方案

如果暂时没有图片，程序会显示占位符方块。可以边玩边截图，逐步添加。
